 Amazon Sales Analysis – Power BI Project

This repository contains an interactive Amazon Sales Analysis Dashboard built in Power BI.
It turns raw sales data into clear, meaningful insights to support business decision-making.

Key Insights

Total sales and revenue analysis

Category-wise and product-wise performance

Monthly and yearly sales trends

Order distribution and customer patterns

KPIs created using DAX

 Files Included

Amazon sales analysis.pbix – Main Power BI report

 Tools & Skills Used

Power BI Desktop

DAX (Data Analysis Expressions)

Data Cleaning & Transformation

Data Modeling

Visualization & Dashboard Design

 Project Purpose

The goal of this project is to analyze Amazon sales data and uncover insights that help track performance, identify trends, and support strategic decisions.

 How to Open the Project

Download the .pbix file from this repo

Open it using Power BI Desktop

Explore dashboard pages, filters, and interactive visuals
